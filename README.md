# About the Project
https://cohort-2-assignment2.netlify.app/

This project is a **modern, responsive landing page** for a SaaS/product platform. It includes the following sections and features:

## 1. Top Navigation Bar
- Logo (video + images)  
- Navigation links (Features, Enterprise, Pricing, Resources)  
- Primary call-to-action buttons (Sign In, Download)  

## 2. Hero Section
- Main headline and short description  
- Email input / CTA button  
- Large illustrative image / background  

## 3. Trusted By / Logos Section
- Row of company logos to showcase credibility  

## 4. Feature Highlights
- Multiple two-column sections  
- Text description paired with visuals or screenshots  
- Highlights product features and benefits  

## 5. Documentation / Content Preview
- Sidebar-style navigation for quick access  
- Main content displayed in card-style layout  

## 6. Intelligent Assistant / UI Preview
- Large UI mockup or interactive preview  
- Accompanying descriptions and feature highlights  

## 7. Enterprise Features Section
- Title and short introductory text  
- Feature blocks covering security, compliance, and scalability  

## 8. Case Studies / Customer Stories
- Card-based layout with images, quotes, and author info  
- Highlights real-world user success stories  

## 9. Recent Highlights / Changelog Section
- Grid of articles/cards  
- Titles, short descriptions, product category, and date  

## 10. Final Call-to-Action Section
- Strong headline encouraging action  
- CTA buttons for engagement (e.g., “Join us →”)  

## 11. Footer
- Multi-column navigation links  
- Company info, legal links, and contact info  

---

## Key Technical Features Implemented
- **Full-width sections** while keeping main content constrained  
- **Sticky headings** for content sections  
- **Accessible and semantic HTML** using `<section>`, `<article>`, `<nav>`, `<time>`  
- Smooth **hover, focus, and interactive states** for buttons and links  
- Proper **grid column alignment** for multi-column layouts  
- Optimized **card and content spacing** using `gap` and padding
